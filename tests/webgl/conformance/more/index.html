Tests for the WebGL canvas context

These tests are intended to serve the following purposes:

  • Assert spec conformance (so that WebGL pages work the same across browsers and hardware)
  • Check the safety of the GL binding (bounds checking, same origin policy)
  • Provide performance numbers for developers

Test runners

  • Run tests manually
  • Run all tests in parallel
  • Run all tests sequentially

Demos

  • OpenGL for the web (requires working FBOs and null textures or using canvases as textures)
  • Green screen video demo with a color remapping filter (requires loading <video> to texture with texImage2D)

Running the tests

  1. Install a browser with WebGL support
  2. Open one of the test runners linked above in your browser.
  3. For more control over the tests, go the GitHub page, see the readme, and clone the repo.
