If statement #3 above is passed, then the canvas should be covered with a flag pattern. The background of the "body" element (which may or may not fully cover the canvas) should be lime. The rules applied are "HTML {background: url(flagbg.gif)}", and "body {background: lime}". All other properties of this rendering should be UA-default.
