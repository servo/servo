Testing Statement #11 Below of Section 7.1 of CSS1 Spec

  • 11. A declaration consists of a property, a colon and a value. Around each of these there may be whitespace. A property is an identifier. Any character may occur in the value, but parentheses, brackets, braces, single quotes and double quotes must come in matching pairs. Parentheses, brackets, and braces may be nested. Inside the quotes, characters are parsed as a string.
