  • 5. The value of a property may be a percentage that refers to another property.

If statement #5 above is passed, this paragraph should have a line-height of 20pt with a font-size of 10pt. The rules applied are "P {font-size: 10pt}" and "P {line-height: 200%}". The 200% refers to font-size, so 200% x 10 = 20. All other properties are UA-default. This emphasized text should have a font size of 200% of the UA default.
