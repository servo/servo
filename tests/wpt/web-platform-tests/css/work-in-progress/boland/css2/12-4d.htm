Testing Fifth Testable Statement of Sec 12.4 of CSS2 Specification

  • 5. Quotation marks are inserted in appropriate places in a document with the "open-quote" and "close-quote" values of the content property.

PASS CRITERIA:

If statement #5 is passed, then there are opening quotation marks on every paragraph in a BLOCKQUOTE, and a single closing quote at the end. Rules applied are "blockquote p:before {content: open-quote}, "blockquote p:after {content: no-close-quote}, and "blockquote p.last:after {content: close-quote}".
