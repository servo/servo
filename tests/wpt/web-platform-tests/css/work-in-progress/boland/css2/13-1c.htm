  • 4. The CSS2 page model specifies how a document is formatted within a rectangular area.

PASS CRITERIA:

If statement #4 above is passed, then the page box size is set of 8.5 x 11 inches and a 4cm margin is created on all sides between the page box edge and the page area. The rule applied is "@page {size 8.5in 11in; margin: 2cm}". All other properties of this page should be UA-default.
