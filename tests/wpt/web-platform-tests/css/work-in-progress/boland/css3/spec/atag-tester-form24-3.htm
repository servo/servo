TEST PURPOSE:

To evaluate an authoring tool's formats according to SC2.4

TEST AGAINST:

Success Criterion 2.4 - Any Web content (e.g., templates, clip art, example pages, etc.) that is bundled with the authoring tool or preferentially licensed to the users of the authoring tool (i.e., provided for free or sold at a discount), must conform to WCAG Level AAA when inserted.
