TEST PURPOSE:

To evaluate an authoring tool's formats according to SC3.2

TEST AGAINST:

Success Criterion 3.2 -
  • The authoring tool must always provide a check (automated check, semi-automated check or manual check) for each applicable requirement to conform to WCAG Level AAA
  • The authoring tool must always infomr the author of any failed check results prior to completion of authoring
