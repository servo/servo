This test passes if you see three gray rectangles with a number of lists inside each, as follows.

The first rectangle should contain a black numbered list, with the numbers in reverse order. After the first item there should a be a blue numbered list, indented and numbered in normal order.

The second rectangle should contain two numbered list. The first one should be black and numbered in reverse order while the second should be blue and numbered in normal order. They should both have the same indentation level.

The third rectangle should contain two black, numbered lists, the first with three items, the second with two items. Both lists should be numbered in reverse.

  1. First reversed list, item 3
    1. First inner list, item 1
    2. First inner list, item 2
  2. First reversed list, item 2
  3. First reversed list, item 1
  1. Second reversed list, item 3
  2. Second reversed list, item 2
  3. Second reversed list, item 1
  1. Second inner list, item 1
  2. Second inner list, item 2
  3. Second inner list, item 3
  1. Third reversed list, item 3
  2. Third reversed list, item 2
  3. Third reversed list, item 1
  1. Final reversed list, item 2
  2. Final reversed list, item 1
