This test passes if there are two gray rectangles: the first with two numbered lists (3 items each), the second with three un-numbered lists (2 items each).
First list of each type should be blue, second list of each type should be black and the third un-ordered list should be green.

Each numbered list should start at 1.
Both the numbers and the list markers should be outside of the gray rectangles.

  1. Ordered list 1, item 1
  2. Ordered list 1, item 2
  3. Ordered list 1, item 3
  1. Ordered list 2, item 1
  2. Ordered list 2, item 2
  3. Ordered list 2, item 3
  • Unordered list 1, item 1
  • Unordered list 1, item 2
  • Unordered list 2, item 1
  • Unordered list 2, item 2
  • Unordered list 3, item 1
  • Unordered list 3, item 2
