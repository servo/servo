This test assumes that the address bar is positioned below the window's title bar, and above the page. If your browser uses an alternative layout, adjust your testing accordingly to ensure that the testing interacts with the edges of the window decoration.

  1. Select some text on this page.
  2. Drag the selection to your browser's title bar.
  3. Drag the selection back to your browser's address field. Fail if the mouse cursor shows that the text cannot be dropped.
  4. Fail if the cursor and/or input focus caret respond in the wrong position (ie. if the drag operation seems to think the address field is offset from its actual location).
  5. Release it. Pass if the selected text appears in the address field.
  6. Repeat the test with other UI text inputs.
