ARIA Notify API Test: Document Announcement

This test checks the behavior of the ariaNotify API with a simple announcement on the document object.

Setup Instructions

Setup: Use a screen reader (NVDA, JAWS, VoiceOver, etc.) or other assistive technology that supports the ARIA Notify API.

Instructions: Click the "Run Test" button and listen for the announced text from your screen reader.

Test: Simple Announcement

Expected: Screen reader should announce "Hello, world!" using default priority (normal).
