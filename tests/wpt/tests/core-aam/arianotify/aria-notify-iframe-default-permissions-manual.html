This test checks the behavior of the ariaNotify API when used within an iframe with default permissions policy settings.

Setup Instructions

Setup: Use a screen reader (NVDA, JAWS, VoiceOver, etc.) or other assistive technology that supports the ARIA Notify API.

Instructions: Listen for the announced text from your screen reader. The test will run automatically when the page loads.

Test: iframe with Default Permissions Policy

Expected: Screen reader should announce "Announcement from iframe with default permissions".
