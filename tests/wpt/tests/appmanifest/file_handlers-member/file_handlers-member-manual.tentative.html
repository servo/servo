File Handling Test

Instructions (details may be OS-dependent)

  1. Install this site as a PWA
  2. Download this file
  3. In your file browser, right click the downloaded file (file_handlers-sample-file.txt)
  4. Select "Open with" > "File Handling WPT"
  5. If your browser prompts you to allow File Handling WPT to open the file, choose allow
  6. The app window that opens should indicate success of this test.
