Features

Strict Privacy

No trackers, fingerprinting, or data collection. Browsing history is not logged at all.

Performance

Built with Rust, Ringtail is memory-safe and optimized.

Zero Telemetry

No usage stats or no crash reports get sent to our servers.

Open Source

Ringtail's souce code is available for free publicly on GitHub. Built upon Servo, an open-source web browser engine.

JavaScript Extension Support (beta)

Add new features to Ringtail with JavaScript-powered extensions.

Simple GUI

Clean interface with no bloat; only what you need, nothing you don't. Fast to load and easy to use.
