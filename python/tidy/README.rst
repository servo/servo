servo_tidy
==========

Servo's code and license checker.

Installation
++++++++++++

Install from PyPI:

::

  pip install servo-tidy

or from git:

::

  pip install -e git+https://github.com/servo/servo.git#egg=servo_tidy&subdirectory=python/tidy

To run the tests
++++++++++++++++

::

  ./mach test-tidy --self-test
